# Demo data

All files in this directory are deterministic synthetic data generated by
`scripts/make_demo_data.py`. They are intended only to exercise file validation,
loading, model forward passes, training loops, checkpointing, and CSV export.

They are not IXI, OASIS, or clinical data and must not be used for scientific
evaluation. Recreate the directory with:

```bash
python scripts/make_demo_data.py --output-dir data/demo --overwrite
```
